# STUNIR Video Tutorials

Welcome to the STUNIR video tutorial series! These tutorials guide you through using STUNIR from beginner to advanced levels.

## How to Use These Tutorials

### Written Scripts
Each tutorial has a written script in this directory that you can follow along with.

### ASCII Cinema Recordings
Terminal recordings are available in the `recordings/` directory. Play them with:

```bash
asciinema play recordings/01_getting_started.cast
```

Or view online at [asciinema.org/~stunir](https://asciinema.org/~stunir) (placeholder)

### Video Hosting
Full video tutorials will be available on:
- YouTube: [STUNIR Channel](https://youtube.com/@stunir) (placeholder)
- Project website: [stunir.dev/tutorials](https://stunir.dev/tutorials) (placeholder)
