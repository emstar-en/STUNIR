# STUNIR C C11 skeleton

THIS IS A SKELETON TEMPLATE (STUB). STUNIR CODEGEN SHOULD REPLACE PROGRAM LOGIC SECTIONS.

## Build

- Using `make`:
  - `make`
  - `./stunir_program`

- Without `make`:
  - `cc -std=c11 -O2 -Wall -Wextra -pedantic -o stunir_program main.c stunir_runtime.c`
  - `./stunir_program`

## Notes
- This skeleton is intentionally tiny and deterministic.
- It is **not** a semantic implementation of STUNIR; it is packaging + stub entrypoints.
